# lftp-client

A small image based in [debian:jessie](https://hub.docker.com/_/debian/) with LFTP client.

Docker hub: [dockito/lftp-client](https://hub.docker.com/r/dockito/lftp-client/)

```shell
docker pull dockito/lftp-client
```
